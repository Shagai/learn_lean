# Algebra

This topic uses the proof tools from the introduction folder to formalize simple algebra. The first lesson stays close to familiar identities; the second widens the view to functions and sets, which are foundational for later algebra and analysis.

## Learning goals

- Recognize algebraic laws already encoded in `mathlib`.
- Distinguish concrete calculations from abstract structure-driven proofs.
- Use set inclusion as a proof problem.
