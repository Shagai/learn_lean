# Calculus

This folder introduces calculus only after the proof foundations are in place. The first lesson uses polynomial derivatives because they are mathematically familiar and the corresponding Lean library theorems are approachable.

## Learning goals

- Read derivative statements written with `deriv` and `HasDerivAt`.
- See how a library theorem and a manual proof can describe the same derivative fact.
- Connect differentiability to continuity.
